# dart_extension

Dart extension is a native Dart extesnions with handy methods.

# Installation

In `pubspec.yaml` file, add this:

```
dart_extension:
  git: 
    url: git://github.com/hhgz9527/dart_extension
```

because this package not upload to pub.dev, so need use git.
